# Paquete Eta4ROc

Este paquete de Julia almacena todas las funciones y métodos necesarios para los cálculos de eta en todos sus métodos de estimación: no paramétrica, paramétrica y kernel.
También incluye métodos para el cálculo de AUC y para el índice de Youden.

Para usarlo en un nuevo script simmlemente se debe importar el paquete: `using Eta4Roc`.
